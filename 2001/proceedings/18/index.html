YAPC::Europe 2001 - Mutagenic modules

This directory contains some supporting materials for the talk Mutagenic modules. It includes a preliminary version of the slides I plan to use, along with some sample code. The examples mostly require Perl 5.6, and one of them also requires Brian Ingerson's Inline module.
