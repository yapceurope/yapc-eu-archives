I think for most of you, this is rather explosive code. In the main package, I first create a dumb object, which has no data and functionality at all. On the moment I call a method on it --here simply print()--, the AUTOLOAD is triggered.

The AUTOLOAD method does two things: load data and assigns functionally by changing the type of the object. The latter is typical Perl, and not possible in any other language I know of.
