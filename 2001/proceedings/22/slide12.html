New Legislation

We now have two new pieces of legislation:

  1. All programmers must be able to use whichever spelling variants they want each time they type a word in a computer program
  2. Programmers must be able to write computer programs in whatever language, dialect or patois they choose

When these become law, all current programming languages become illegal

We need to find ways for Perl to comply with these laws
