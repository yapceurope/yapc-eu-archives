Writing Docs

Scrap the traditional boilerplate 

        ############################################
        # Procedure - my_foo
        #
        # Inputs - string $yarblockos, integer $bar
        # Outputs - float
        ############################################

Docs should flow as a descriptive narrative. 

        =item B<checkout>

            my $receipt = $basket->checkout($credit_card);

        The items in a customer's shopping $basket (a Shop::Basket) are
        checked out and paid for on their $credit_card (Shop::Card)
        and an order placed with our warehouse.  Returns a $receipt
        (Shop::Sale) containing details of this sale.

        If the $credit_card is not valid or doesn't have enough
        credit, false will be returned.

        The $basket will be emptied and ready for more shopping.
