Clear Interfaces

KEY CONCEPT HERE

If you get the interface right, everything else is gravy.

You can always change the implementation, the interface is forever.

  • DO ONE THING! Do it well.

  • Think about who will use it, when and how.

        my @foo = @{ bar() };

  • Does it mesh well?

        pod2pdf infile outfile

                instead of

        pod2pdf infile > outfile

                makes this common Unix idiom difficult

        pod2pdf infile | xpdf
