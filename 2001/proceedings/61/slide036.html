Specific Golfing Nits

  • Use an explicit return() to avoid confusion.

  • Don't smash logic together. Do one thing per line

        foo(++$i) if $i;

  • Avoid chaining complex operations.

        map map grep map grep for 1..10;

  • Use intermediate (temporary) variables for clarity

  • Throw logical bits into a function.
