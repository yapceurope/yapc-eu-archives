Version Control

KEY CONCEPT HERE

You need some sort of central code repository. Doesn't matter what system you use.

  • Serves as an automatic, incremental backup

  • Reduces feelings of code ownership

  • Everyone's on the same page.

  • Provides a way out of mistakes

  • No mystery files Foo.pm.jon3

  • Provides lots of nice statistics
