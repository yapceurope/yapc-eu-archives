Bad Interfaces

A lot of popular code has a baaaaaad interface. Bad interfaces mean confusion. Confusion means bugs.

  • localtime in Perl parrots localtime in C

    my($mday, $mon, $year) = (localtime)[3..5];
    printf "%d/%d/%d", $mday, $mon + 1, $year + 1900;

        should probably have been

    my %time = localtime;
    printf "%d/%d/%d", @time{qw(mday mon year)};

    and then there's the y2k problem...
