The next pages shall visit some of the facilities offered by Tk. Tk defines widgets (window gadgets, window thingies), which represent screen-items.

The list of widgets is enormous: there are some simple ones, like Button and Label. But some are really complex, like getOpenFile, which is a file selector.

All standard widgets are demonstrated by the widget program, which is part of the Perl/Tk distribution.
