Parse Trees

  • Another way of creating parse tree is by using auto action.
  • If the global variable $::RD_AUTOACTION is set, the string contents is interpretated as an action, and added to each production that does not already end in an action.
  • $::RD_AUTOACTION = q {[@item]}; creates very simple parse trees. 
In [1] := 3d4
Out [1] = $VAR1 = [
          'start',
          [
            'die',
            [
              'count',
              [
                'number',
                3
              ]
            ],
            'd',
            [
              'faces',
              [
                'number',
                4
              ]
            ]
          ],
          ''
        ];
