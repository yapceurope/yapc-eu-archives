PerlPoint: present and document with Perl

Lorenz Domke, Pixel GmbH
Jochen Stenzel, Pixel GmbH

The documentation contains an article, an introduction, a reference, FAQ's and more, all in source and in HTML format. A README file is provided to introduce this package. The bundle includes a preinstalled version of PerlPoint which should run out of the box (on UNIX systems) to enable CD users to play with the software without further installations.

A snapshot of PerlPoint::Converters which is not available on CPAN but is the base of the software provided with the doc package.

The current version of PerlPoint::Package, which is the same version as found on CPAN.
