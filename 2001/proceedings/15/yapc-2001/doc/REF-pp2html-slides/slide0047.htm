U

The \U tag marks text to be formatted as underligned.

Syntax

\U<body>

Options

No options are allowed. Using options will cause a syntax error.

Body

The body is the string to be formatted.

Example 


  This is very \U<important>.

produces 


  This is very important.

See also

\SUB, \SUP, \F
