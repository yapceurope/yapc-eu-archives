Paragraphs

The following paragraph types are used by pp2html:

  • Comments paragraphs
  • Headline paragraphs
  • Text paragraphs
  • Block paragraphs
  • Verbatim block (here documents)
  • List paragraphs
  • Table paragraphs
  • Condition paragraphs
  • Variable assignment paragraphs
  • Macro paragraphs
