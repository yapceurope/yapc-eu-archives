Embedding PerlPoint into PerlPoint

This is just for fun. Set the "lang" option to "pp" to try it: 


  Perl Point \EMBED{lang=pp}can \EMBED{lang=pp}be
  \EMBED{lang=pp}nested\END_EMBED\END_EMBED\END_EMBED.
