SUB

The \SUB tag marks text to be formatted as subscript.

Syntax

\SUB<body>

Options

No options are allowed. Using options will cause a syntax error.

Body

The body is the string to be formatted.

Example 


  S = sum(A\SUB<i>)  (i = 1, ..., 8)

produces 


  S = sum(Ai)  (i = 1, ..., 8)

See also

\SUP, \U, \F
