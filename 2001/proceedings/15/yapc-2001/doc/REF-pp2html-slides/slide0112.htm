Cache

To speed up the processing of large documents, PerlPoint provides a caching mechanism. If your input file is named xyz.pp, the PerlPoint Parser will create a cache file called .xyz.pp.ppcache. To take advanage of this feature you must specify the --cache option because caching is not enabled by default. The cache file can be cleared with the --cacheCleanup option.
