SUP

The \SUP tag marks text to be formatted as superscript.

Syntax

\SUP<body>

Options

No options are allowed. Using options will cause a syntax error.

Body

The body is the string to be formatted.

Example 


  E = m c\SUP<2>

produces 


  E = m c2

See also

\SUB, \U, \F
