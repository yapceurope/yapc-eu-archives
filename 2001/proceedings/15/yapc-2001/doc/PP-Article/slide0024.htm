6.2 pp2latex and pp2sdf

pp2latex and pp2sdf can be used for the preparation of printable output formats. PerlPoint file which have been written for pp2html should be readable by pp2latex and ppsdf.

pp2latex is still relatively inflexible concerning document classes. It is possible to supply a template file which determines the document class and provides a certain setup, but there are not many examples. If you use images you should supply an .eps file for each .gif or .jpeg file. pp2latex will then automatically inlucde these Postscript files in the LaTeX output. For controlling the size, you can use the epsfxsize or epsfysize parameter in the \IMAGE tag: 


 \IMAGE{epsfxsize=14cm src="./images/PerlPoint-Structure.gif"}
