Colors in Tables

Question:

How can I use color in tables?

Answer:

Use the \TABLE tag with special options: 


 \TABLE{separator="," bgcolor="#CCCCCC" head_bgcolor=yellow}
   Column 1 , Column 2
   value 1  , value 2
   value 3 , value 4
 \END_TABLE

Discussion:

The above examples yields:

Column 1Column 2
value 1value 2
value 3value 4
